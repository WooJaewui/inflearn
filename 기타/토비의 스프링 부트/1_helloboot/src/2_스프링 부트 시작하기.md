> ## HTTP 요청과 응답

### 참고
- 요청이 없는 응답이 없다


### HTTP 프로토콜
- 응답에 대한 요청 규칙을 정의해 놓은 것


### Request
- Request Line : Method, Path, HTTP Version
- Headers
- Message Body


### Response
- Status Line : HTTP Version, Status Code, Status Text
- Headers
- Message Body
